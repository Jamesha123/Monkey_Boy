# Monkey Boy

2D adventure game rebuilt from `MonkeyBoy.jar`.

## Project layout

- `src/` — Java source code
- `res/` — game assets (sprites, maps, sounds, fonts) extracted from the JAR
- `bin/` — compiled classes (generated by `build.bat`)
- `MonkeyBoy.jar` — original runnable JAR
- `config.txt` — fullscreen and volume settings

## Requirements

- JDK 17+ (Temurin recommended)

## Build and run

```bat
build.bat
run.bat
```

Or from the project root:

```bat
java -cp "bin;res" main.Main
```

## VS Code

Use the **Run Monkey Boy** launch configuration in `.vscode/launch.json`.

## Notes

- Save files are stored in your user home directory as `MonkeyBoy_save.dat`.
- Resources load from the `res/` folder on the classpath, matching the original JAR layout.
